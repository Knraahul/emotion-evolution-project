# Emotion Evolution in Stories

AI-powered emotion analysis system for stories, PDFs, images, and text using NLP, OCR, sentiment analysis, and interactive visualization.

---

## Features

- Text, PDF, and image input support
- OCR-based text extraction
- Emotion and sentiment analysis
- Peak emotion detection
- Interactive visualizations
- Chapter-wise analysis
- AI-generated summaries
- Streamlit web application

---

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- NLTK
- VADER Sentiment
- OCR
- Transformers
- OpenCV
